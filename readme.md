> SOme module to play with npm publishing


## Install

```
$ npm install --save jfmengels-playground
```


## Usage

```js
const jfmengelsPlayground = require('jfmengels-playground');

jfmengelsPlayground('unicorns');
//=> 'unicorns & rainbows'
```
